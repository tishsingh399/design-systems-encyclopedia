# Design Systems Encyclopedia

Searchable encyclopedia of 16+ design systems with alias-aware semantic search across components, foundations, patterns, templates, and resources.

## Features

- **Alias-aware search** — Search "badge" and find Badge, Chip, Pill, Lozenge, Tag, Label across all systems
- **Semantic concepts** — Search "progressive disclosure" and find Accordion, Collapse, Drawer, Details, etc.
- **16 design systems** — Material, Carbon, Atlassian, Spectrum, Fluent, Ant, Polaris, Lightning, GOV.UK, Primer, PatternFly, Paste, Gestalt, USWDS, Chakra, Orbit
- **Direct links** — Every result links to the specific component/pattern doc page, not just the system homepage
- **35+ resources** — Token tools, AI UX patterns, directories, Figma collections, pattern libraries
- **Category filters** — Components, Foundations, Patterns, Templates with counts

## Tech Stack

- React 18 + Vite
- Zero dependencies beyond React
- Deployed on Vercel

## Development

```bash
npm install
npm run dev
```

## Deploy

Push to GitHub → Connect to Vercel → Auto-deploys on every push.
